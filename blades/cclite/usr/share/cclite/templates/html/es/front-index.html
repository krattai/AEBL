Cclite Quickstart

Instalador Web | Inicio de sesión de usuario | Administración |
  1. Uso el análisis de instalación para analizar el estado actual de la instalación
  2. Uso el instalador para configurar los valores de configuración, base de datos usuario y contraseña
  3. Crear un registro (trading group) for your system, for example, Dalston , Newington , Islington Un registro es una base de datos mysql...
  4. Iniciar sesión como administrador Inicie sesión en su registro como Gerente con contraseña Gerente
  5. (when Manager) Set up a currency or two
  6. Crear usuarios o que vengan Crear usuarios y deje el comercio.
  7. Recuerde quitar o deshabilitar al instalador y cambiar las contraseñas de administrador y sysaccount. Quitar esta página también!
Los trabajos por lotes como escritura rss, leer entrada de transacciones por lotes y gráficas son ahora en la página de administración para el administrador. Pueden ejecutar directamente desde la web sin usar cron. Por cron, unas pequeñas modificaciones deben hacerse en las secuencias de comandos.
