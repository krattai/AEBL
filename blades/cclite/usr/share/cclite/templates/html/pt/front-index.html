Cclite Quickstart

Instalador da Web | Logon do usuário | Administração |
  1. Uso a análise de instalação para analisar a situação atual da instalação
  2. Uso o instalador para configurar valores de configuração, senha e usuário de banco de dados
  3. Criar um registro (trading group) for your system, for example, Dalston , Newington , Islington Um registro é um banco de dados mysql...
  4. Faça logon como gerente Logon para o seu registro como gerente com senha gerente
  5. (when Manager) Set up a currency or two
  6. Criar usuários ou deixá-los vir Criar usuários e deixá-los a negociar.
  7. Lembre-se de remover ou desabilitar o instalador e alterar as senhas de sysaccount e admin. Remova esta página também!
Trabalhos em lotes como escrever rss, ler a entrada de transação em lotes e gráficos estão agora na página de gestão para o administrador. Eles podem ser executado diretamente da web sem usar o cron. Por cron, algumas modificações precisam ser feitas nos scripts.
