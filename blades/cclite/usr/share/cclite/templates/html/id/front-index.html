Cclite Quickstart

Web Installer | Logon pengguna | Administrasi |
  1. Penggunaan Analisis instalasi menganalisa keadaan saat ini instalasi
  2. Penggunaan installer mengatur nilai-nilai konfigurasi, database pengguna dan password
  3. Membuat registry (trading group) for your system, for example, Dalston , Newington , Islington Registry adalah satu database mysql...
  4. Logon sebagai manajer Logon ke registri Anda sebagai Manajer dengan password Manajer
  5. (when Manager) Set up a currency or two
  6. Membuat pengguna atau membiarkan mereka datang Membuat pengguna dan membiarkan mereka perdagangan.
  7. Ingatlah untuk menghapus atau menonaktifkan installer dan mengubah sysaccount dan admin Password. Menghapus Halaman ini juga!
