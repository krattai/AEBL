Alat validasi formulir ini dapat memeriksa diperlukan, bidang bilangan bulat, email dan tanggal in your forms. It is easily portable to all your forms and will help save server side resources from form field validation and page reloads. All the messages are customized in the hidden form element where you also select the validation requirement. You can mix the required element with any of the other elements. This code is completely portable and can be used with any form by just adding your hidden elements appropriately and adding the onClick command to a button.

Mengikuti aturan.
1) Untuk mengatur penggunaan diisi nama field yang ada dan menambahkan r_ ke awal bidang nama dan membuat elemen tersembunyi.
CONTOH:
<input type="text" name="your_fieldname">
<input type="hidden" name="r_your_fieldname" value="Required error message goes here">

2) Untuk mengatur integer bidang menggunakan nama field yang ada dan menambahkan i_ ke awal bidang nama dan membuat elemen tersembunyi.
CONTOH:
<input type="text" name="your_fieldname">
<input type="hidden" name="i_your_fieldname" value="Integer error message goes here">

3) Untuk menetapkan tanggal bidang penggunaan nama field yang ada dan menambahkan d_ ke awal bidang nama dan membuat elemen tersembunyi. Ini field tanggal yang hanya menerima dalam MM/DD/YY format atau M/D/YY.
CONTOH:
<input type="text" name="your_fieldname">
<input type="hidden" name="d_your_fieldname" value="Date error message goes here">

4) To validate an email field use your existing field name and add an e_ to the beginning of the field name and create a hidden element.
CONTOH:
<input type="text" name="your_fieldname">
<input type="hidden" name="e_your_fieldname" value="The email address format is incorrect">

Inisialisasi script.
Anda harus menggunakan jenis = "tombol" bukan tipe = "Kirim" untuk tombol SUBMIT, javascript mengirimkan menyerahkan untuk Anda.
CONTOH:
<input type=button Value="Button Message Here" onclick="check(form,form.elements.length)">

Sampel ini menggunakan semua alat-alat validasi tiga dalam kombinasi.

Alamat email: (required and email validation)
First Name: (required validation)
Nama: (required validation)
Alamat 1: (required validation)
Alamat 2: (no validation)
Berapa usiamu: (required and integer validation)
Bertahun-tahun telah Anda telah script: (integer validation)
Apa itu hari ini tanggal: (mm/dd/yy) (date validation)
Pilih subjek: (required validation)
Pilih tombol Radio: tes 1 menguji 2 menguji 3 (required validation)
Pilih Semua bahwa berlaku: tes 1 menguji 2 menguji 3 (required validation on box 1)
