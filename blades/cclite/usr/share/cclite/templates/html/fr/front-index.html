Programme d'installation Web | Ouverture de session utilisateur | Administration |
  1. Utilisation l'analyse de l'installation d'analyser l'état actuel de l'installation
  2. Utilisation le programme d'installation pour configurer les valeurs de configuration, mot de passe et utilisateur de la base de données
  3. Créer un registre (trading group) for your system, for example, Dalston , Newington , Islington Un registre est une base de données mysql...
  4. Ouverture de session dans le gestionnaire Connexion à votre base de Registre comme gestionnaire avec mot de passe gestionnaire
  5. (when Manager) Set up a currency or two
  6. Créer des utilisateurs ou laissez-les venir Créer des utilisateurs et leur faire le commerce.
  7. N'oubliez pas de supprimer ou de désactiver le programme d'installation et de changer les mots de passe sysaccount et admin. Supprimer cette page aussi !
Lots par exemple rss d'écriture, lecture des entrées de transaction par lots et la représentation graphique sont maintenant sur la page de gestion de l'administrateur.
