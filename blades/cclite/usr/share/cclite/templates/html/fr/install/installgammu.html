Vous pouvez utiliser des transactions de sms avec plusieurs services d'enregistrement dans la version actuelle, mais celui-ci est par défaut, voir le manuel ! Le Registre et monnaie doivent déjà exister .

Vous devez également configurer /etc/gammu-smsdrc for the phone or dongle that you are using and for inbox/outbox etc. See the example in the config qui est pour le nom du répertoire = Huawei 1750 donnant un raccordement style. C'est le matériel utilisé pour le test de la version actuelle.

Vous devrez peut-être également utiliser le Utilitaire Switch mode Si un dongle de téléphone se déclare comme un périphérique de stockage
