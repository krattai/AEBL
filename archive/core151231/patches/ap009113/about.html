About the AEBL Platform

AEBL began being promoted as a Mobile Media device, which is correct. What is important to express is that AEBL is more than just a media device. It is, formost, a technology framework which has been built to serve and enable individuals, organizations, and communities with important tools that support this end.

The AEBL project is located on the AEBL github project page.

The AEBL wiki is located on the AEBL github wiki page.

Reporting Problems

Please use the AEBL issues forum located on the AEBL issues page to report bugs in your AEBL system. However, check the existing issues as well as closed issues before reporting a new bug, as it may already be reported or resolved.

Github does not have a forum feature, although github users will use issues for discussions. While it works, it may not be easy to navigate and find what you are looking for. The AEBL project is mirrored on sourceforge, which does have a traditional forum. If you want to ask a question, get clarification, join the project, and/or provide other feedback, please visit the AEBL discussions on sourceforge.
