באפשרותך להשתמש ב- sms עסקאות עם קבלו מספר במהדורה הנוכחית, אבל זה ברירת מחדל, עיין במדריך! את רישום ומטבע חייב להיות קיים .

עליך גם לקבוע את תצורת /etc/gammu-smsdrc for the phone or dongle that you are using and for inbox/outbox etc. See the example in the config ספריית המשנה שהיא עבור שם = 1750 Huawei נותן חיבור בסגנון AT. זהו החומרה המשמשות לבדיקת הגירסה הנוכחית.
