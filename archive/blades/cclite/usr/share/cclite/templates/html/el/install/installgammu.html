Θα πρέπει επίσης να ρυθμίσετε τις παραμέτρους /etc/gammu-smsdrc for the phone or dongle that you are using and for inbox/outbox etc. See the example in the Config υπο-κατάλογο που είναι για όνομα = Huawei 1750 δίνοντας μια σύνδεση στυλ AT.
